Repository overview

The repositories page of ScenarioRunner shows you all known repos

The view can be sorted by URI or by date of the last run

From here you can go into the details for a specific measurement, or compare them.

Comparing is possible for different repos, different branches and different runs as long as they are on the same machine. Also different machines are supported if you compare the same repo.

Currently we do not support comparing different machines AND different repos at the same time. The tool will notify you if you attempt to do so :)
